# AndroidWorld: A Dynamic Benchmarking Environment for Autonomous Agents

## 元信息
- arXiv ID: 2405.14573
- 作者: Christopher Rawles, Sarah Clinckemaillie, ... Timothy Lillicrap, Oriana Riva (Google Research / DeepMind)
- 类型: academic-paper (mobile GUI agent benchmark)

## 规模
- **116 个**程序化任务 (programmatic tasks)
- **20 个**真实 Android 应用
- 任务用自然语言参数化表达，可无限变体生成

## 硬数字（关键）
- 原论文发布时，最佳 agent 任务完成成功率仅 **30.6%**
- 截至 2026（二手综述）：SOTA agent 在 AndroidWorld 116 任务上已超 **90%**，基准趋于饱和
- 对比：把桌面 web agent 直接迁移到移动端效果较差

## 与综述关联
AndroidWorld 是当前移动 GUI agent 最主流的在线基准；其"30.6% → 90%+"两年间的跃升，量化了"agentic OS / 手机自动化"从不可用到接近可用的演进速度。
