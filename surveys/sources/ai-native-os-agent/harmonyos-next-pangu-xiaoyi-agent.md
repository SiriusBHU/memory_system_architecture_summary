# HarmonyOS NEXT + 盘古大模型 5.0 + 小艺系统级智能体

（来源：深圳湾 shenzhenware.com 对 HarmonyOS NEXT 发布的报道，及 mydrivers/腾讯新闻等二手报道；HDC 2024，2024-06）

## AI 原生定位
- HarmonyOS NEXT（纯血鸿蒙 / HarmonyOS 5）将华为 AI 融入操作系统层
- 集成华为自研**盘古大模型 5.0 (PanGu 5.0)**
- 小艺 (Xiaoyi/Celia) 从语音助手升级为**系统级智能体 (system-level agent)**
- 小艺集成在屏幕底部导航栏，可随时唤起，支持拖拽输入文本/图片/文档

## 小艺智能体能力（厂商自报硬数字）
- 智能感知 **23 大场景**
- 提供 **300+ 项关键服务**
- 任务推理与规划成功率 **90%**
- 支持 **23 种常用记忆类型**，万亿 tokens 知识
- 复杂能力示例：把含表格的图片转为电子表格；根据邮件内容规划导航路线

## 隐私
- 模型与敏感数据本地化，不上传云端
- 端云同构 (端侧与云侧 isomorphic)

## HDC 2024 开发者侧
- 开放意图框架，赋能第三方 app 构建智能体（出行专家、办公助手、娱乐专家、生活助手、运动专家等）
- 端云协同推理：app 把结构化元数据传给小艺，按预设诊断 prompt 与领域需求分析
