# Personal LLM Agents: Insights and Survey about the Capability, Efficiency and Security

## 元信息
- arXiv ID: 2401.05459
- 主要作者: Yuanchun Li 等 25 位作者（含清华/北京等机构，含华为/小米等产业背景作者）
- 版本: v1 (2024-01-10), v2 (2024-05-08)
- 分类: cs.HC

## 核心愿景
- "Personal LLM Agents will become a major software paradigm for end-users"
- 个人 LLM agent 深度集成个人数据与设备，超越传统智能助手
- 把 LLM agent 定位为下一代端侧/OS 级智能助手

## 三大维度
1. Capability：意图理解、任务规划、工具使用、个人数据管理
2. Efficiency：资源受限环境（端侧）下的性能优化
3. Security：个人数据保护与 agent 行为安全

## 与综述关联
学术界对"端侧个人 LLM agent 作为新软件范式/OS 级助手"的系统综述，
与 Apple/Google/Huawei 把系统级 agent 内置到 OS 的产业实践直接呼应；
其 capability/efficiency/security 三分法可作为对比三家 agentic OS 的分析框架。
